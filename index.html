Proxy Worker URL
Browsers block direct API calls (CORS). A tiny free Cloudflare Worker sits in the middle and forwards requests.

Setup takes ~2 minutes:
1. Go to workers.cloudflare.com → free account
2. Create Worker → paste the code shown below → Deploy
3. Copy the worker URL (e.g. https://espresso-proxy.YOUR.workers.dev) and paste it here
Worker Code (copy → paste into Cloudflare)
export default {
  async fetch(request) {
    if (request.method === 'OPTIONS') {
      return new Response(null, { headers: {
        'Access-Control-Allow-Origin': '*',
        'Access-Control-Allow-Methods': 'POST,OPTIONS',
        'Access-Control-Allow-Headers': 'Content-Type',
      }});
    }
    const { apiKey, ...body } = await request.json();
    const res = await fetch('https://api.anthropic.com/v1/messages', {
      method: 'POST',
      headers: {
        'Content-Type': 'application/json',
        'x-api-key': apiKey,
        'anthropic-version': '2023-06-01',
      },
      body: JSON.stringify(body),
    });
    const data = await res.json();
    return Response.json(data, { headers: {
      'Access-Control-Allow-Origin': '*'
    }});
  }
};
Anthropic API Key
Required for AI-powered Dial In estimates.
Get your key at console.anthropic.com → API Keys.
Your key is saved in this browser (localStorage) and never leaves your device.
Your key is never sent anywhere except directly to api.anthropic.com.
It is stored in sessionStorage and cleared when you close the tab.
